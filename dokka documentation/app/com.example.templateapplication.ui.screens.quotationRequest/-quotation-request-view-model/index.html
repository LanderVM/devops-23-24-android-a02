QuotationRequestViewModel

class QuotationRequestViewModel(restApiRepository: ApiRepository, googleMapsRepository: GoogleMapsRepository) : ViewModel

ViewModel for handling quotation request-related functionalities.

Manages the data and operations for creating and processing quotation requests. Interacts with REST API for quotation data and Google Maps API for location services. It maintains the state of various API calls and manages the UI state for quotation requests.
