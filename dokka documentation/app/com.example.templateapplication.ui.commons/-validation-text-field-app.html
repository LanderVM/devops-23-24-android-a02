ValidationTextFieldApp

@Composable
fun ValidationTextFieldApp(placeholder: String, text: String = "", onValueChange: (String) -> Unit = {}, keyboardType: KeyboardType = KeyboardType.Text, imeAction: ImeAction = ImeAction.Done, errorMessage: UiText? = null, isError: Boolean = false, isVisible: Boolean = false, singleLine: Boolean = false, maxLines: Int = 1)

Composable function for a validation text field with customizable features.

This function creates an outlined text field that supports validation and error handling. It provides various customization options like keyboard type, IME action, error message, visibility toggle, and leading icon.

Parameters

modifier

Modifier to be applied to the text field for customization.

placeholder

Text to be displayed when the text field is empty.

text

Current text value of the text field.

onValueChange

Callback function for text value change events.

keyboardType

Type of keyboard to be used for input.

imeAction

IME action for the keyboard.

errorMessage

Message to be displayed in case of an error.

isError

Boolean flag indicating whether an error state should be shown.

isVisible

Boolean flag for visibility control, particularly useful for password fields.

singleLine

Boolean flag to set the text field as a single line input.

maxLines

Maximum number of lines for the text field.
